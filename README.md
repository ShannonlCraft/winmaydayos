# Shannon Craft Website (No Coding Required)

If you don't know how to code, that's okay. Use these exact steps.

## 1) Open the website on your computer

1. Open this folder.
2. Double-click `OPEN_ME_FIRST.html`.
3. Click the page you want to view.

If needed, you can also double-click `index.html` directly.

## 2) Edit simple text (optional)

If someone helps you edit later, these are the main files:

- `index.html` = Home page
- `about.html` = About page
- `reflections.html` = Reflections page
- `download.html` = Free ebook page

## 3) Put it online (easiest method)

Use Netlify Drag-and-Drop:

1. Go to https://app.netlify.com/drop
2. Drag this whole folder into the browser window.
3. Netlify gives you a live link automatically.

## 4) What to customize first

- In `download.html`, replace the `Download Ebook` button link (`href="#"`) with your real ebook URL.
- Replace placeholder text in `about.html` and `reflections.html` when ready.
